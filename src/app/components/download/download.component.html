All data displayed on the site is open source and you can download it by visiting the data GitHub repository:
https://github.com/narmafraz/ThaqalaynData

The link above contains the data per page of the site, if instead you would like the all data for one book then that can be found in: https://github.com/narmafraz/ThaqalaynData/books/complete

Also note that this data was generated from other raw data and the script used to generate it is in: https://github.com/narmafraz/ThaqalaynDataGenerator

If this data is useful to you and you do end up creating a project using it then we would like to hear about it please!
You can do this by contacting us through raising an issue on one of the above repositories
and feel free to raise a Pull Request for us to link back to your project!

We hope that we can grow this data over time, hopefully with your help.
